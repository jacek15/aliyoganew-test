# Ali Yoga Studio (Next.js)

This is the Next.js + Tailwind + Framer Motion recreation of the static HTML site.

## Run locally

```bash
cd web
npm install
npm run dev
```

## Configure canonical URLs / sitemap host

Set your deployed site URL (used for `metadataBase`, `sitemap.xml`, and `robots.txt`):

```bash
NEXT_PUBLIC_SITE_URL="https://your-domain.com"
```

## Assets

- Fonts are loaded from `public/fonts/` (copied from the original `../fonts/` folder).
- Image placeholders are in `public/images/`. Replace them with your real images (keep the same filenames or update the `src` paths).
